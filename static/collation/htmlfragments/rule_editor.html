Batch upload global rules

Global rules can be pre-loaded from a csv file with tab separation. The first row is assumed to be headers and will be ignored.

The columns should be

  1. original form
  2. normalised form
  3. rule class (must match one of your rule classes)
  4. ignore supplied tags (Y/N)
  5. ignore unclear tags (Y/N)
  6. only apply to nomina sacra (Y/N)
